# Portofoliu Proiecte IT - Raul

Acest repository conține o selecție de proiecte dezvoltate în timpul studiilor de Licență și Master, acoperind domenii precum Game Development, Machine Learning și Securitate Cibernetică.

## Structura Proiectelor

### 1. Game Development & Scripting (Godot / Lua)
* **Simulator de Grădinărit:** Proiect de licență dezvoltat în **Godot Engine**. Include animații originale, logică de interacțiune și mecanici de gameplay (GDScript).
* **Automatizare Boss Roblox:** Script complex în **Lua** pentru automatizarea mecanismelor de luptă și pattern recognition într-un mediu virtual.

### 2. Machine Learning & AI (Python)
* **Detectare Dialecte:** Implementarea unui model de Procesare a Limbajului Natural (NLP) pentru identificarea variațiilor lingvistice.
* **Reconstrucție Imagini (MLP):** Utilizarea rețelelor neuronale de tip Multi-Layer Perceptron pentru restaurarea și de-noising-ul imaginilor.
* **Studiu Modele Clasice:** Optimizarea modelelor SVM, Random Forest și ANN prin Grid Search și analiza densității seturilor de date.

### 3. Full-Stack & Security Research (PHP / C++ / HTML / CSS)
* **Sistem de Autentificare:** Interfață web stilizată integrată cu backend PHP pentru procesarea datelor.
* **Analiză Hash-uri:** Modul dezvoltat în **C++** pentru compararea hash-urilor de parole, utilizat în scopuri de cercetare a vulnerabilităților de stocare.
